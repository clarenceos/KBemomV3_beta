I got tired of paying subscriptions for something as simple as a timer.

Kettlebell EMOMs are one of the most effective training formats:
simple structure, measurable volume, and brutally effective conditioning.

This app focuses on doing one thing extremely well:

**A fast, reliable, kettlebell-focused EMOM timer that stays out of your way.**

No ads.
No subscriptions.
No analytics.
No accounts.

Just you, your kettlebell, and a timer that works.

# Features

## Workout Builder

* 30+ preloaded kettlebell exercises
* Add unlimited custom exercises
* Favorites system for fast access
* Set reps and weight per exercise
* Drag reorder exercises
* Multi-set support (repeat circuits automatically)
* Save and reload workouts instantly

## Complex Builder (v3.x)

* Create kettlebell complexes (2‚Äì5 movements)
* Shared weight across movements
* Save custom complexes to your personal library
* Quick edit or full edit anytime
* Works seamlessly inside normal workouts

## Timer Engine

* Wall-clock based timing (not interval-based)
* Accurate even when switching apps or locking phone
* Dynamic countdown ring visualization
* 3-2-1 start countdown
* Skip forward or backward rounds instantly
* Double-tap pause/resume
* Wake Lock support to prevent screen sleep
* Haptic feedback on supported devices
* Freestyle mode for open EMOM sessions

## Logging & History

* Automatic workout logging
* Stores exercises, reps, weight, rounds, volume, and duration
* Expandable log entries
* Delete logs anytime
* Copy workout summary for sharing

## Stats Dashboard

* Lifetime volume tracking
* Per-exercise volume and rep tracking
* Sortable stats
* Training streak tracking

*Stats are approximate and intended for personal reference and motivation only.*

## Settings

* Accent color customization
* Sound selection
* Mute option
* Persistent preferences

## Platform & UX

* Add to Home Screen support (iOS and Android)
* Works fully offline
* No installation required
* No internet required after loading
* Single-file architecture for maximum portability

# How to Use

1. Select an exercise or complex
2. Set reps and weight
3. Tap **+** to add it to your workout
4. Repeat until your circuit is complete
5. Adjust sets if needed
6. Press **START**
7. Train

Everything else is automatic.

* Single HTML file
* React 18 via CDN

* Wake Lock API support
* Web Audio API for timer sounds
* No backend
* No cloud storage
